# Personal Website

Welcome to my personal website! This project showcases my portfolio, experience, and skills, providing a one-stop destination for learning more about me. It is built using modern web development technologies, with attention to design and usability.

---

## Features

- **Responsive Design**: Optimized for desktops, tablets, and mobile devices.
- **Dynamic UI**: Engaging animations and interactivity implemented using JavaScript.
- **Custom Styling**: Styled with SCSS for efficient and maintainable CSS.
- **Professional Showcase**: Filled with content directly from my resume, highlighting projects, skills, and achievements.

---

## Built With

### Languages and Technologies
- **HTML5**: Structuring the content of the website.
- **CSS3**: Styling and layout.
- **JavaScript**: Adding interactivity and functionality.
- **Sass (SCSS)**: Preprocessor for CSS to write modular and reusable styles.
